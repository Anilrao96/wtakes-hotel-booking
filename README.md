# wtakes - Hotel Booking Website

## Deploy to Vercel (Free)

### Option 1: Deploy via Vercel CLI (Recommended)

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to project folder:
```bash
cd d:\Booking.com-clone
```

3. Deploy:
```bash
vercel
```

4. Follow the prompts:
   - Login with GitHub/GitLab/Email
   - Confirm project settings
   - Get your live URL!

### Option 2: Deploy via Vercel Website

1. Go to https://vercel.com
2. Sign up/Login (free account)
3. Click "Add New" → "Project"
4. Import this folder or connect GitHub
5. Deploy!

### Option 3: Deploy to Netlify (Alternative)

1. Go to https://app.netlify.com/drop
2. Drag and drop the entire `Booking.com-clone` folder
3. Get instant URL!

### Option 4: Deploy to GitHub Pages

1. Create GitHub repository
2. Push code to GitHub
3. Go to Settings → Pages
4. Enable GitHub Pages
5. Access via: https://yourusername.github.io/repo-name

## Quick Deploy Commands

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
cd d:\Booking.com-clone
vercel --prod
```

Your website will be live at a URL like: https://wtakes.vercel.app
